Grasp the Future

Kennesaw State University Senior Design Final Project.
Team Members : Joel Mathew Philip, Diego Escobar Escobar, Jeremy Plattel, Timothy Lemley

* Best KSU Senior Design Mechatronics Project 2021.
* 2nd Place Overall KSU Engineering Expo 2021

System Overview

- The robot and conveyor system has been upgraded with industry 4.0 concepts in mind. It is now a perfect demonstration of concepts like IIOT, Remote control, Machine vision, Deep Learning, and network based integration.

- The system can find payloads on the ground and discern their exact position at runtime. It can then grasp them and sort them into different conveyors based on their type.

- This is useful for order fulfillment centers such as Amazon or Walmart. It could also be useful in places like airport luggage systems.

Machine Vision and Learning


- Machine Vision : Determine the number of payloads in the workspace, the position ( X,Y ) of the closest payload and the rotation of the closest payload.

- For Machine Vision, a program is written to draw a red box around the boxes and find it's centroid. Using this the above data types are found. The robot uses this data to pick up boxes according to the closest distance from home position.

- Machine Learning : Determines the type of the box using a Neural Network. Model trained using approximately 5000 images taken over the course of the semester.

- Machine Learning Steps :
1) Define hyper-parameters to make a functional AI model
2) Create a model that accepts 30X30 pixel images and processes them through one 2000 node hidden layer
3) Define the different output classes. For this project the five different classes were Black box, Styrofoam box, Cardboard box, Clear box, and Null or “It doesn’t know”
4) After defining the model, it must be trained. Training is the process of changing the weights of each node over and over again until the model accuracy is high enough.
5) The weights are initially set to random values and then updated based on a concept called ‘gradient descent.’ Between each change, thousands of training images will be passed through the model.


SCADA Application


- SCADA(Supervisory control and data acquisition) is a computer system architecture that provides a graphical interface between a user and an industrial system.

- These SCADA applications provide some level of control over the system and also provides graphical access to diagnostics and status information for devices in the system or the system as a whole.

- CONTROL :
1) Process start, stop, and reset
2) Manual control of whether high or low pressure is used to grip a box.

- VIEW :
1) System and safety status info
2) Active system alarms
3) Pressure level in the gripping mechanism
4) Whether a payload is present in the gripper
5) Whether the gripper is actuated
6) Machine vision and machine learning data for a payload such as type, position, orientation, and whether a lid is required.
