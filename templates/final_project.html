System Overview

- The robot and conveyor system has been upgraded with industry 4.0 concepts in mind. It is now a perfect demonstration of concepts like IIOT, Remote control, Machine vision, Deep Learning, and network based integration.

- The system can find payloads on the ground and discern their exact position at runtime. It can then grasp them and sort them into different conveyors based on their type.

- This is useful for order fulfillment centers such as Amazon or Walmart. It could also be useful in places like airport luggage systems.


SCADA Application


- Supervisory control and data acquisition or SCADA is a computer system architecture that provides a graphical interface between a user and an industrial system.

- These SCADA applications provide some level of control over the system and also provides graphical access to diagnostics and status information for devices in the system or the system as a whole.

- CONTROL : 1) Process start, stop, and reset 2) Manual control of whether high or low pressure is used to grip a box.

- VIEW : 1) System and safety status info 2) Active system alarms 3) Pressure level in the gripping mechanism 4) Whether a payload is present in the gripper 5) Whether the gripper is actuated 6) Machine vision and machine learning data for a payload such as type, position, orientation, and whether a lid is required.
